# Security Policy

## Supported Versions

This is an academic project and only the latest default branch is considered supported for security fixes.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

1. Open a private security advisory on GitHub, or
2. Contact the maintainer directly through the profile listed in the project metadata.

Please include:

- A clear description of the issue.
- Reproduction steps or proof of concept.
- Potential impact.
- Suggested remediation if available.

We will acknowledge receipt as soon as possible and coordinate a responsible disclosure timeline.
